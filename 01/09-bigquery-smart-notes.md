# 🧠 BIGQUERY (SMART NOTES)

## 🔑 Rule
👉 **Serverless + SQL + Analytics = BigQuery**

---

## ⚙️ WHAT IS BIGQUERY
👉 Fully managed data warehouse  
👉 No infrastructure management  
👉 SQL-based querying  

---

## ✔️ FEATURES
- ✔️ Serverless (no setup, no servers)
- ✔️ Cost-effective (pay per query)
- ✔️ Massive scale (TBs–PBs data)
- ✔️ Fast analytics (columnar storage)

---

## 🎯 USE CASE
- data analysis  
- reporting  
- dashboards  
- business intelligence  

---

## ❌ NOT FOR
- ❌ transactional systems (OLTP)
- ❌ real-time updates (limited)
